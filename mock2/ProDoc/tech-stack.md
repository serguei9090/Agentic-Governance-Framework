# Tech Stack (LogVibe Analyzer)

- **Frontend:** React (Vite+TypeScript), TailwindCSS, Framer Motion.
- **Backend:** FastAPI, Python 3.10.
- **Log Processing:** Drain3 (Clustering), ChromaDB (Vector Search).
- **Architecture Standard:** AVAS (Agentic Visual Architecture Standard).
- **Testing:** Vitest (Unit), Playwright (E2E).
