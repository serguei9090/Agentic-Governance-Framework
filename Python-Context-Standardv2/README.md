# **Enterprise AI-Agent Boilerplate (Python V2)**

**Status:** Production Ready (Level 5)
**Version:** 2.0.0
**Architect:** Antigravity Agent

## **Overview**
This repository contains the **Golden Standard** for initializing AI-driven Python software projects. It is the Python counterpart to the React Context Standard, adapted for modern Python toolchains (Reflex, Ruff, MyPy).

## **Core Capabilities**
1.  **Strict Governance:** Agents cannot write code without checking `agentContext.md` first.
2.  **Enterprise Frameworks:** Python-specific governance (Ruff, MyPy, Pytest).
3.  **Self-healing:** Includes "ProDoc" protocols for self-maintaining documentation.
4.  **Operational:** Includes "Agentic Workflows" for Dependency Updates, Migrations, and Security Audits.
5.  **Flexible:** Built for Modular Monoliths using Reflex and FastAPI.

## **Quick Start**
To apply this governance to a new project:

1.  Copy the `AgentSetUpProject` folder to your new repo.
2.  **Review the Checklist:** Open `startAgentProjectSetUp.md` and check/uncheck optional protocols.
3.  Run the Agent Directive: "Initialize Project Context using `AgentSetUpProject/agentContext.md`".
4.  The Agent will scaffold your `.agent/rules/` and file structure automatically.
