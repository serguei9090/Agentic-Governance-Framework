# Diagramming Standards

**Purpose:** Instructions for generating diagrams.

## Tool Standard
*   **Tool:** Mermaid.js
*   **Usage:** Embed directly in markdown files using ```mermaid blocks.

## Diagram Types
1.  **Sequence Diagrams:** For API flows and interaction between User, UI, and Backend.
2.  **Class Diagrams:** For Core Logic and Python Data Models.
3.  **Entity-Relationship (ER):** For Database Schema verification.
